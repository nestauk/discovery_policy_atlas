# Experimental Components

Wireframes and prototypes for UI interventions table.

## Policy Dashboard Preview

A prototype interventions explorer with:
- Intervention theme list with impact/evidence filtering
- Different interventions within each theme 
- Study outcomes drill-down

### Run

```bash
cd frontend
npm run dev
```

### View

http://localhost:3000/policy_dashboard_preview

```
components/experimental/PolicyDashboardPreview.tsx  # Main component
```
